Ας δουλέψουμε με αριθμούς.

Σου δίνεται ένας πίνακας αριθμών (πραγματικών). Σκοπός είναι να υπολογίσεις τη διαφορά ανάμεσα στο μικρότερο και το μεγαλύτερο στοιχείο του. Η συναρτηση που θα φτιάξεις θα δέχεται έναν οποιονδήποτε αριθμό παραμέτρων. Σε περίπτωση που η λίστα παραμέτρων είναι κενή, η συνάρτηση θα επιστρέφει τον αριθμό 0.

Οι δεκαδικοί αριθμοί στον υπολογιστή αναπαριστώνται σαν κλάσματα με βάση το 2 (δυαδικά). {% if "python 2" in interpreter|lower %} (διάβασε περισσότερα εδώ). {% else %} (διάβασε περισσότερα εδώ). {% endif %} Επομένως, θα εξετάσουμε το αποτέλεσμα με ακρίβεια ±0.001.
Σκέψου πώς θα δουλέψεις με {% if "python 2" in interpreter|lower %} έναν οποιονδήποτε αριθμό παραμέτρων . {% else %} έναν οποιονδήποτε αριθμό παραμέτρων. {% endif %}

Είσοδος: Ένας οποιοσδήποτε αριθμός παραμέτρων σε μορφή ακέραιων ή πραγματικών αριθμών.

Έξοδος: Η διαφορά ανάμεσα στο μικρότερο και το μεγαλύτερο με μορφή ακέραιου ή πραγματικού αριθμού.

Παράδειγμα: 

checkio(1, 2, 3) == 2
checkio(5, -5) == 10
checkio(10.2, -2.2, 0, 1.1, 0.5) == 12.4
checkio() == 0

Πώς χρησιμοποιείται: Εδώ θα μάθεις πώς να περνάς έναν απροσδιόριστο αριθμό παραμέτρων σε μια συνάρτηση.
