Заданий масив чисел (float або / і int). Вам потрібно знайти різницю між найбільшим (максимум) і самим малим (мінімум) елементом. Ваша функція повинна вміти працювати з невизначеною кількістю аргументів. Якщо аргументів немає, то функція повертає 0 (нуль).

Числа з плаваючою точкою представлені в комп'ютерах як двійковий дріб {% if "python 2" in interpreter|lower %} (прочитати можна тут). {% else %} (прочитати можна тут). {% endif %} Результат перевіряється з точністю до третього знаку, тобто з точністю ±0.001.

Зауваження: Прочитайте про те, як працювати {% if "python 2" in interpreter|lower %} зі змінним числом аргументів. {% else %} зі змінним числом аргументів. {% endif %}

Вхідні дані: Змінне число аргументів як числа (int, float).

Вихідні дані: Різниця між максимумом і мінімумом як число (int, float).
